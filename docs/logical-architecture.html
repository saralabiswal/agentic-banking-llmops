L1 Context Assembly ≤ 200ms
Parallel async fetch from all source adapters (150ms hard timeout each). Graceful degradation — source failure marks partial_context, never fails the pipeline. Unified CustomerProfile written to Valkey with EX=300, NX=True.
asyncio.gather Valkey TTL Pydantic v2 Feature Store
session_id → profile key
L2 Vector Search ≤ 80ms
Dynamic query built from live profile signals. Hybrid dense + BM25 retrieval merged with RRF. Cross-encoder reranks top-20 → top-3 policy chunks returned. KB version tracked in audit record.
sentence-transformers rank_bm25 Qdrant RRF merge
policy chunks
L3 Multi-Agent Orchestration ≤ 15s
Hub-and-spoke orchestrator — agents never call each other directly. Tool authorization enforced in code before any call. Agents PROPOSE only. Schema-validated outputs. Failures route to HumanReviewQueue.
LiteLLM / Ollama Tool registry Valkey checkpoint Pydantic outputs
proposed actions
L4 Guardrails & Policy ≤ 150ms
REGULATORY → BUSINESS POLICY → RESPONSIBLE AI in strict sequence. Regulatory block stops all further checks. Flagged actions enter SLA-backed approval queue. Rule versions recorded for replay.
YAML rule store BISG fairness Approval queue hot-reload
approved actions
L5 A/B Evaluation & Model Gov. ≤ 10ms
Hash-based deterministic variant assignment. Outcomes feed back into experiment results. Champion/Challenger (5% traffic). Three-type drift detection: feature (KS), prediction (PSI), performance (recall 30d).
MLflow Evidently scipy stats hash bucket
variant-tagged actions
L6 SDK Surface & Execution ≤ 20ms
Blueprint catalog for product teams. Channel adapters: push, SMS, CRM. Delivery receipts + async outcome capture. Outcomes feed back to L5. trace_id threads from context assembly through enrollment.
